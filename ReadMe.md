# Prompt Engineering In Action (LLM Use Cases)

## Обзор проекта
Структурированный репозиторий для экспериментов и практических реализаций различных сценариев использования больших языковых моделей (LLM) с акцентом на промт инжениринг и локальное выполнение.

## [Проект: Rental Ads Analyzer](rental_ads/ReadMe.md)

### Цель
Автоматизированный анализ текстовых объявлений об аренде жилья для извлечения структурированной информации о количестве людей, ищущих жилье, с использованием локальных LLM через Ollama.

### Технологический стек
- **Python 3.9+**
- **Ollama** (локальные LLM модели: llama3, mistral, или специализированные)
- **Pandas** для обработки и анализа данных
- **LangChain** (опционально) для продвинутых цепочек промтов

### Основной функционал
1. **Загрузка и предобработка данных**
   - Чтение объявлений из CSV

2. **Извлечение информации с помощью LLM**
   - Использование промтов для определения количества ищущих аренду
   - Валидация извлеченных данных

### Планы развития проекта
1. **Добавление новых сценариев использования LLM:**
   - `docs_difference/`    - Анализ документов и выделение различий
   - `trend_analyse/`      - Сравнение и анализ рынка
   - `content_generation/` - Генерация контента по шаблонам  
  
2. **Улучшение существующего функционала:**
   - Добавление веб-интерфейса (Streamlit/Gradio)
   - Интеграция с базами данных
   - Реализация асинхронной обработки
   - Добавление кэширования промтов  
  
3. **Эксперименты с промт инженирингом:**
   - Сравнение различных шаблонов промтов
   - Few-shot и zero-shot подходы
   - Цепочки промтов (prompt chaining)
   - Автоматическая оптимизация промтов
